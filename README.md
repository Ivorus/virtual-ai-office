## 📖 О проекте

**Virtual AI Office** — десктопное приложение для Windows и Linux, которое превращает работу с ИИ в визуальный и интерактивный опыт. Вместо того чтобы просто набирать промты в чат, ты видишь живой офис: 6 агентов сидят за столами, встают чтобы выпить кофе, ходят на совещания — и при этом выполняют реальные рабочие задачи через Anthropic API.

> 💡 Идея: что если управление ИИ-командой было бы похоже на SimCity, а не на чат-бот?

### ✨ Что нового в v5
- 🔧 **6 агентов** вместо 4: добавлены Макс (DevOps) и Ана (Аналитик/Автор)
- 🚀 **Claude Sonnet 4.6** — новейшая модель (или Haiku 4.5 / Opus 4.6 на выбор)
- 📝 **4096 токенов** на ответ вместо 1024 — агенты пишут подробнее
- 🧠 **Память агентов** — помнят контекст последних 3 разговоров
- 📋 **Копировать результат** — одна кнопка на карточке задачи
- 📤 **Экспорт** всех результатов в папку
- ⌨️ **Ctrl+Enter** — быстрая отправка задачи
- 💰 **Правильный расчёт цены** для каждой модели

## ✨ Возможности

### 🤖 Живые ИИ-агенты
- **4 специалиста** с уникальными характерами и системными промтами
- Анимированные пиксель-арт человечки с ходьбой, морганием, выражением лица
- Мысли в облачках над головой — видишь что думает каждый агент
- Автономное поведение: сами ходят пить кофе, на перерыв, в туалет

### 🏢 Визуальный офис
- Нарисованный вручную 2D офис с комнатами и мебелью
- **6 тем оформления**: Ночной офис, Светлый лофт, Зелёный Bio, Корпоратив, Тёмный Luxury, Cyberpunk
- **10 видов мебели**: столы, диваны, кофемашина, растения, переговорный стол, серверная стойка, доска и др.
- Перетаскивание мебели мышью в режиме редактирования
- Перекраска любого предмета в произвольный цвет

### 🏗️ Небоскрёб
- Добавляй неограниченное количество этажей
- Каждый этаж — отдельная комната со своей мебелью и темой
- Отправляй агентов на нужный этаж одной кнопкой
- Переключайся между этажами в реальном времени

### 👑 Система одобрения
- Агенты **никогда** не действуют без твоего разрешения
- Каждый ответ показывается с кнопками **Разрешить / Отклонить**
- Полный контроль над тем что делает каждый агент

### 📊 Аналитика и отчёты
- Подсчёт стоимости каждого запроса к API в реальном времени
- История всех задач в SQLite базе данных
- Вкладка отчётов с таблицей задач и статистикой расходов

### 📋 Редактор быстрых команд
- Добавляй, редактируй и удаляй шаблоны команд
- Быстрый доступ к часто используемым задачам одним кликом

## 📦 Установка

### Вариант 1 — EXE файл (рекомендуется)

```bash
# 1. Скачай репозиторий или ZIP
# 2. Распакуй в любую папку
# 3. Двойной клик на:
build_exe.bat
```

Скрипт автоматически установит все зависимости и создаст `VirtualAIOffice.exe` на рабочем столе.

### Вариант 2 — Запуск через Python

```bash
# Требования: Python 3.11+

# Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/virtual-ai-office.git
cd virtual-ai-office

# Установить зависимости
pip install -r requirements.txt

# Запустить
python main.py
```

Или просто двойной клик на `run.bat`.

## ⚙️ Настройка

### Получить Anthropic API ключ

1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйся (можно через Google)
3. Перейди в **Billing** → пополни баланс от $5
4. Перейди в **API Keys** → **Create Key**
5. Скопируй ключ — он показывается только один раз!

### Ввести ключ в приложение

В запущенном приложении нажми **⚙️ Настройки** → вставь API ключ → OK.

> 💰 Стоимость: один ответ агента ≈ $0.001–0.005 (меньше цента)

## 🚀 Быстрый старт

```
1. Запусти приложение (run.bat или VirtualAIOffice.exe)
2. Нажми ⚙️ Настройки → введи Anthropic API ключ
3. Выбери агента в выпадающем списке
4. Напиши задачу или выбери быструю команду
5. Нажми 🚀 Поставить задачу
6. Дождись ответа → нажми ✅ Разрешить или ❌ Отклонить
```

## 🗂️ Структура проекта

```
virtual-ai-office/
├── main.py              # Главное приложение, UI, логика задач
├── office_scene.py      # Визуализация офиса, анимация людей, мебель
├── requirements.txt     # Python зависимости
├── run.bat              # Быстрый запуск через Python
├── build_exe.bat        # Сборка EXE для Windows
├── upload_to_github.bat # Загрузка проекта на GitHub
└── README.md            # Документация
```

После первого запуска появятся файлы данных:

```
├── settings.json        # API ключ (в .gitignore)
├── floors.json          # Расположение мебели и этажи
├── quick_commands.json  # Быстрые команды
├── office.db            # История задач (SQLite)
└── office_cloud/        # Результаты работы агентов (txt)
```
